# Installation 
### 1] Terminal :

    git clone https://github.com/faizan619/ReactVite-TodoApp.git
    npm install
    npm run dev

### 2] Github CLI :
    gh repo clone faizan619/ReactVite-TodoApp
    npm install
    npm run dev

### 3] Download the Zip File
    extract file
    npm install
    npm run dev
